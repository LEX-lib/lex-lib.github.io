  <behavior>
    Meeting mapper:
    - mapToCreateMeeting with duration_unit undefined → result.duration_unit === 'minutes'
    - mapToCreateMeeting with duration_unit 'hours' → result.duration_unit === 'hours'
    - mapToCreateMeeting with duration_minutes 60 + duration_unit 'hours' → result preserves both
    - mapToCreateMeeting with description undefined → result.description is undefined
    - mapToUpdateMeeting excludes id, created, updated, collectionId, collectionName, date
    - mapToUpdateMeeting preserves title, duration_minutes, duration_unit, description
    - mapFromRecordMeeting with duration_unit undefined → result.duration_unit === 'minutes'
    - mapFromRecordMeeting with duration_unit 'hours' → result.duration_unit === 'hours'

    Task mapper:
    - mapToCreateTask preserves date, title, jira_link, description
    - mapToCreateTask with jira_link undefined → result.jira_link is undefined
    - mapToUpdateTask excludes id, created, updated, date
    - mapToUpdateTask preserves title, jira_link, description
    - mapFromRecordTask returns a spread of the record (no transformation)

    Support mapper:
    - mapToCreateSupport preserves date, title, link, description
    - mapToCreateSupport with link undefined → result.link is undefined
    - mapToUpdateSupport excludes id, created, updated, date
    - mapToUpdateSupport preserves title, link, description
    - mapFromRecordSupport returns a spread of the record (no transformation)

    Day-status mapper:
    - mapToCreateDayStatus preserves date and status
    - mapToCreateDayStatus output does NOT contain id, created, updated
    - mapFromRecordDayStatus returns a spread of the input record

    useDurationField composable:
    - seed(90, 'hours') → unit.value === 'hours', enteredValue.value === 1.5, durationMinutes.value === 90
    - seed(30, 'minutes') → unit.value === 'minutes', enteredValue.value === 30, durationMinutes.value === 30
    - seed(undefined, 'minutes') → enteredValue.value === null, durationMinutes.value === undefined
    - seed(undefined, undefined) → unit.value === 'minutes' (default)
    - Setting enteredValue.value = 2 when unit = 'hours' → durationMinutes.value === 120
    - Setting enteredValue.value = 45 when unit = 'minutes' → durationMinutes.value === 45
    - Setting enteredValue.value = null → durationMinutes.value === undefined
    - seed(120, 'hours') after initial seed(30, 'minutes') → unit changes to 'hours', enteredValue becomes 2
    - fractionDigits.value === 2 when unit === 'hours'
    - fractionDigits.value === 0 when unit === 'minutes'

    Constants:
    - DSU_DAY_STATUS_VALUES has exactly 5 elements
    - DSU_DAY_STATUS_VALUES includes 'sl', 'vl', 'holiday', 'bl', 'others'
    - DSU_DAY_STATUS_LABELS['sl'] === 'Sick Leave'
    - DSU_DAY_STATUS_LABELS['vl'] === 'Vacation Leave'
    - DSU_DAY_STATUS_LABELS['holiday'] === 'Holiday'
    - DSU_DAY_STATUS_LABELS['bl'] === 'Birthday Leave'
    - DSU_DAY_STATUS_LABELS['others'] === 'Others'
    - DSU_DAY_STATUS_SHORT_LABELS['sl'] === 'SL'
    - DSU_DAY_STATUS_SHORT_LABELS['vl'] === 'VL'
  </behavior>
