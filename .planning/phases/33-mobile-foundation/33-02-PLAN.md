```ts
export function useIsMobile(breakpoint = 639): Ref<boolean> {
  const query = window.matchMedia(`(max-width: ${breakpoint}px)`)
  const isMobile = ref(query.matches)            // seeds initial value synchronously (avoids M-6 race)
  const handler = (e: MediaQueryListEvent) => { isMobile.value = e.matches }
  onMounted(() => query.addEventListener('change', handler))
  onUnmounted(() => query.removeEventListener('change', handler))
  return isMobile
}
```

| Viewport / display-mode media queries → reactive refs | Environmental signals (read-only) cross into composable state |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-33-05 | Elevation of Privilege | Auto-triggering the install prompt without user gesture | mitigate | `.prompt()` is intentionally NOT called in this phase; Phase 37 wires it behind an explicit user click. Grep guard: useMobileEnv.ts and App.vue contain no `.prompt(`. |
