Conditional geocoder override (verbatim from 21-CONTEXT.md §Specific Ideas D-09) — apply ONLY if geocoder input is illegible on dark:

```vue
<style scoped>
:global(.my-app-dark) .leaflet-control-geocoder-form input {
  background-color: var(--color-surface-card);
  color: var(--color-typo-body);
  border-color: var(--color-surface-divider);
}
</style>
```

Note: Vue 3 SFC `:global(...)` is the canonical "scoped CSS that pierces into teleported/external DOM" syntax. The Leaflet geocoder injects its DOM outside Vue's component scope, so a normal scoped rule wouldn't reach it.
