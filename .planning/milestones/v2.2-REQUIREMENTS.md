## v2.2 Requirements (All Complete)

### Membership Card Toolbar

- [x] **ORG-01**: User can type into a search input to filter the membership card grid in real time by card name or issuer (case-insensitive, partial match); when no cards match, an informative empty-state message replaces the grid; the search can be cleared via a × button
- [x] **ORG-02**: User can select a sort mode from a dropdown to reorder the membership card grid; available modes: Name A–Z, Issuer A–Z, Expiry Date (soonest first — cards without an expiry date sorted last), Recently Added; the selected mode is retained for the current session

**Validation notes:**
- ORG-01 — Delivered in Phase 16 Plan 02: `displayedMemberships` computed filters `records` by `card_name`/`issuer` (case-insensitive partial match); no-results empty state with "Clear search" button; × clears `searchQuery` ref via InputIcon slot in WallecxToolbar
- ORG-02 — Delivered in Phase 16 Plans 01–02: `membershipSortOptions` (4 modes) passed to WallecxToolbar via required prop; `sortMode` ref persisted to `sessionStorage` with whitelist validation on restore
